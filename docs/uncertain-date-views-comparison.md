# 不确定日期事件 - 视图对比

## 显示效果对比表

| 事件类型 | 年视图 | 月视图 | 周视图 | 日视图 | 列表视图 |
|---------|--------|--------|--------|--------|----------|
| **精确日期** | ✅ 正常显示 | ✅ 正常显示 | ✅ 正常显示 | ✅ 正常显示 | ✅ 正常显示 |
| **日期待定** | ✅ 虚线+斜纹 | ✅ 月初显示 | ❌ 隐藏 | ❌ 隐藏 | ✅ 带📅图标 |
| **季度待定** | ✅ 季初显示 | ❌ 隐藏 | ❌ 隐藏 | ❌ 隐藏 | ✅ 带📅图标 |
| **年度待定** | ✅ 年初显示 | ❌ 隐藏 | ❌ 隐藏 | ❌ 隐藏 | ✅ 带📅图标 |

```
┌─────────────────────────────────────────────────┐
│  2024年                                         │
├───────────┬───────────┬───────────┬───────────┤
│  一月     │  二月     │  三月     │  四月     │
│           │           │┌────────┐│           │
│           │           ││ 春季活动││           │
│           │           ││ (待定) ││           │
│           │           │└────────┘│           │

│           │           │  ^斜纹背景│           │

```

**特点**:
- 在对应月份显示
- 虚线边框 + 斜纹背景
- 右上角 📅? 图标
- 蓝色渐变背景

### 📊 周视图 (Week) 和日视图 (Day)

```
周视图 - 3月第一周
┌────┬───────────────────────┐
│    │  3月1日  3月2日  ...  │
├────┼───────────────────────┤
│09:00                      │
│    │                       │
│10:00  [确定事件]          │
│    │   10:00-11:00         │
│11:00                      │
│    │                       │
└────┴───────────────────────┘

⚠️ 日期待定事件不显示
   (避免与精确时间事件混淆)
```

**设计理由**:
- 周/日视图主要用于查看**精确时间安排**
- 显示"日期待定"事件会造成误导
- 用户可切换到月视图查看待定事件

## 实际场景示例

### 场景1：春季活动规划

```json
{
  "title": "春季运动会",
  "datePrecision": "month",
  "approximateMonth": "2024-04",
  "content": "具体日期根据天气情况决定"
}
```

**显示位置**:
- ✅ 年视图: 4月格子中
- ✅ 月视图: 4月1日带虚线边框
- ❌ 周/日视图: 不显示

---

### 场景2：年度目标

```json
{
  "title": "完成课程体系改革",
  "datePrecision": "year",
  "approximateMonth": "2024-01",
  "content": "2024年内完成"
}
```

**显示位置**:
- ✅ 年视图: 1月显示（年初）
- ❌ 月视图: 不显示
- ✅ 列表视图: 显示为"2024年（具体时间待定）"

## 用户操作流程

### 创建待定事件

1. 点击日历任意位置
2. 选择"日期精确度" → "日期待定"
3. 选择大致月份（如"2024-03"）
4. 填写其他信息并保存

### 查看待定事件

1. **年视图**: 一目了然看到全年待定事项
2. **月视图**: 查看当月待定事项（显示在1号）
3. **列表视图**: 按时间顺序查看所有事项

### 确定日期后

1. 编辑事件
2. 将"日期精确度"改为"精确日期"
3. 设置具体 `startTime` 和 `endTime`
4. 保存后自动移到精确日期位置

## CSS 样式参考

### 虚线边框 + 斜纹背景

```css
.event-uncertain {
  border-style: dashed;
  border-width: 2px;
  opacity: 0.85;
}

.event-uncertain::before {
  background: repeating-linear-gradient(
    45deg,
    transparent,
    transparent 8px,
    rgba(255, 255, 255, 0.15) 8px,
    rgba(255, 255, 255, 0.15) 16px
  );
}
```

### 颜色方案

- **日期待定**: `#3b82f6` (蓝色)
- **季度待定**: `#a855f7` (紫色)
- **年度待定**: `#6b7280` (灰色)

## 总结

| 优势 | 说明 |
|------|------|
| 🎯 **清晰标识** | 虚线+斜纹一眼可辨 |
| 🎨 **视觉分离** | 不同颜色区分精确度 |
| 📱 **响应式** | 各视图适配良好 |
| 🔄 **灵活切换** | 可随时确定具体日期 |
| 🚀 **性能优化** | 索引支持快速查询 |
